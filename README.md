# PWHT Typst Template

> [!IMPORTANT]
> In den Kapiteln treten Syntaxfehler auf, wenn der Live-View nicht aktiv ist.

## Quickstart

> [!TIP]
> VS Code `Tinymist Typst` Extension installieren

1. Repository klonen
2. Metadaten anpassen
3. Kapitel bearbeiten bzw. hinzufügen
4. Live-Vorschau in VS Code starten
5. PDF exportieren über VS Code

## Funktionsweise

Schau' dir die vorhandene Platzhalter-Dateien an, um zu sehen, wie das Template zu benutzen ist. Für indirekte Verweise, verwende `#vgl`. Konsultiere sonst die [Typst Dokumentation](https://typst.app/docs/).
